# Recruitment Funnel Analysis

## Project Overview

This project analyzes the recruitment funnel of a company to understand how candidates move through different hiring stages.

## Recruitment Stages

* Applications Received
* Candidates Shortlisted
* Candidates Interviewed
* Offers Made
* Candidates Joined

## Dataset

The dataset contains 10,000 job applicants.
The "Best Match" column was used to identify shortlisted candidates.

## Tools Used

* Microsoft Excel
* Data Analysis
* Funnel Visualization

## Key Insight

Out of 10,000 applicants, only a portion of candidates were shortlisted and moved to the next stages.
The funnel helps identify where the biggest candidate drop happens during recruitment.
